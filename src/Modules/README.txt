This folder contains the core modules for the Chronomancer's Paradox mod:

- ConfigManager.cs: Centralized configuration access
- PositionSnapshot.cs: Data structure for position/health history (Task 2)
- TemporalRecall_Data.cs: Spell definition (Task 8)
- TemporalRecall_Logic.cs: Spell casting behavior (Task 9)
- PositionTracker.cs: Network component for tracking and recall (Task 3+)
